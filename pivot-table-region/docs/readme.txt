Pivot Table Plugin
==================

This is a Region plugin that displays a Pivot Table based on a SQL query, with export and charting options.

It is basically an Oracle APEX wrapper for the Javascript Pivot Table by Nicolas Kruchten (https://github.com/nicolaskruchten/pivottable).

For details, see https://ora-00001.blogspot.com/2018/02/pivot-table-plugin-for-apex-updated.html and https://ora-00001.blogspot.com/2013/10/pivot-table-plugin-for-apex.html
